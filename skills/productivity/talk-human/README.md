# talk-human

把技術內容改寫成更易讀、更精簡、也更適合人閱讀的說明。

## 這個 skill 做什麼

這個 skill 用來把原本太技術、太冗長、太抽象的內容，收斂成比較容易理解、比較容易採取行動的版本。

它特別適合用在：
- 技術說明太硬
- 寫法太像內部筆記
- 雖然正確，但不夠好讀
- 需要把內容改寫成比較適合直接貼給使用者的版本

## 什麼時候適合使用

當你需要：
- 把技術段落改得更 human-readable
- 把說明壓短，但又不失去主路徑
- 把抽象分析改寫成更貼近實際使用者情境的說法

這是一個 user-invoked skill。適合在你已經有一份內容，但想把它改得更好讀時使用。
